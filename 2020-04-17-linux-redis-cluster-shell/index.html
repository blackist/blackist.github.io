参照 Redis 主从集群及自启动配置 实现自动安装，首先判断是否已安装，若存在 Redis 服务程序则跳过，否则安装。

通过 if [ ! -f "/usr/local/bin/redis-server" ] 判断 server 程序是否存在，若存在则认为 Redis 已安装。
