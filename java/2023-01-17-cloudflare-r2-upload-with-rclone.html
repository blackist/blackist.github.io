使用 rclone 命令行上传文件到 Cloudflare R2 存储

ref: Upload files to cloudflare R2 with rclone

rclone

First of all, install rclone on our pc.

rclone list file error

Configure rclone

create a rclone.conf file in the path of ~/.config/rclone/.

Contents of the conf are as follows:

  • [r2]: 自定义的名称，当前所配置的存储服务的别名，后续的文件操作需要使用
  • type = s3: 文件操作 API 的类型，R2 API 支持 S3 标准协议
  • provider = Cloudflare: 存储服务提供商 ，通过 man rclone 命令查看文档 Supported providers
  • access_key_id: 创建具有编辑权限的 R2 API 令牌
  • secret_access_key: 同上
  • endpoint: The url that rclone uses to operate files. To get the account id on the top-right of R2 homepage.

Usage

run rclone lsf r2: to see your buckets and rclone lsf r2:my-bucket to show file list within a bucket.
